# Ariston Publications Portal

A modern web application for Ariston Publications, showcasing academic and research publications across various disciplines.

## Features

- Modern, responsive design
- Book showcase with hover effects
- FAQ section
- Newsletter subscription
- Manuscript submission system

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- pnpm (preferred package manager)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/DawoodSaeed/bookworm-portal-glow.git
cd bookworm-portal-glow
```

2. Install dependencies:

```bash
pnpm install
```

3. Start the development server:

```bash
pnpm dev
```

4. Build for production:

```bash
pnpm build
```

## Technology Stack

- React
- TypeScript
- Tailwind CSS
- Shadcn/ui
- Vite
